# 🌍 ALTNEWZ - Where Headlines Meet the Multiverse

What if the news was *wildly* different? Ever wondered what happened in the timeline where squirrels became the dominant species? Or where cats actually landed on the moon first? **ALTNEWZ** is here to answer these questions you didn't know you had.

Feed it a real news headline, watch the AI magic happen, and get 5 hilariously creative alternate timeline stories that'll make you question reality—or at least make you laugh so hard your coffee comes out your nose.

## Why ALTNEWZ?

🎬 **Entertainment on Steroids** - Perfect for procrastination, entertaining friends, or existential crises at 2 AM.

✍️ **Writer's Goldmine** - Stuck on creative ideas? Need inspiration for fiction, worldbuilding, or just general absurdist humor? ALTNEWZ generates endless material to spark your imagination.

🚀 **AI-Powered Creativity** - The intersection of "What if?" and "LOL what?!"

## Getting Started

Ready to explore alternate realities? Here's how:

```bash
# Clone the repository
git clone https://github.com/suyogpoudel/altnewz.git
cd altnewz

# Install dependencies
npm install
# or
pnpm install
# or
yarn install
# or
bun install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and start generating chaos.

## How It Works

1. **Enter a headline** - Any real news story will do
2. **Click generate** - The AI takes over from here
3. **Enjoy the madness** - 5 alternate timeline versions appear
4. **Share with friends** - Watch them lose their minds (in the best way)
5. **Use for inspiration** - Steal ideas for your stories, scripts, games, whatever
